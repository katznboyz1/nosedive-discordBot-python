NOSEDIVE.py

A discord bot that moderates using user ratings.

You most likely came to this site to view the bot's commands, here they are:

@nosedive - This will return the link to this website, for finding the commands.
@nosedive kill - This will turn off the bot, but this only works if you are a verified bot developer/admin.
@nosedive profile - This will display your profile in the discord app.
@nosedive profile [userID] - This will display the user's profile.
